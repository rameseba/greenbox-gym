# GreenBox Gym - Sitio Web Oficial

Repositorio oficial del sitio web de **GreenBox Gym**, Valencia, Venezuela.

## ✨ Características

- 🎨 **Diseño Moderno:** Estética inspirada en Apple con modo claro y oscuro dinámico.
- 📱 **Totalmente Responsivo:** Optimizado para dispositivos móviles y escritorio.
- 🕒 **Sección de Horarios:** Visualización clara de las clases y sesiones.
- 📍 **Integración con Google Maps:** Ubicación exacta con enlace directo.
- 💬 **Contacto Vía WhatsApp:** Integración directa para inscripciones.
- 🎭 **Animaciones Fluidas:** Uso de Framer Motion para entradas y transiciones.

## 🛠️ Tecnologías Utilizadas

- **React 19**
- **Vite** (Build Tool)
- **Tailwind CSS 4** (Estilos)
- **Motion** (Animaciones)
- **Lucide React** (Iconografía)

## 🚀 Instalación y Desarrollo

Sigue estos pasos para ejecutar el proyecto localmente:

1. **Clonar el repositorio:**
   ```bash
   git clone <url-del-repositorio>
   cd react-example
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
   El sitio estará disponible en `http://localhost:3000`.

## 📦 Producción y Despliegue

Para generar la versión de producción:

```bash
npm run build
```

Esto creará una carpeta `dist/` que puedes subir a cualquier servicio de hosting estático como:

- **Vercel** (Recomendado)
- **Netlify**
- **GitHub Pages**

## 📝 Personalización

Puedes editar los datos principales en `/src/App.tsx` modificando las constantes al inicio del archivo:
- `WHATSAPP_NUMBER`
- `EMAIL_CONTACT`
- `INSTAGRAM_HANDLE`
